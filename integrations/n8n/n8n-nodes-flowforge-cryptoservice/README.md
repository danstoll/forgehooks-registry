# n8n-nodes-flowforge-cryptoservice

n8n community node for FlowForge crypto-service.



## Installation

### Community Nodes (Recommended)

1. Go to **Settings** > **Community Nodes**
2. Click **Install**
3. Enter `n8n-nodes-flowforge-cryptoservice`
4. Agree to the risks
5. Click **Install**

### Manual Installation

```bash
npm install n8n-nodes-flowforge-cryptoservice
```

Then restart n8n.

## Configuration

1. Add FlowForge credentials:
   - Go to **Credentials** > **Add Credential**
   - Search for "FlowForge"
   - Enter your API Key and Base URL

2. Use the node in your workflows
